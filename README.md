# VSDX2BPMN

VSDX2BPMN project allows you to convert VSDX to BPMN XML.

## How to Use It

### Download

1. Make sure you have .net 8.0 installed. If not: [.NET 8.0](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
2. Clone this repo.

### Build

1. Open project with sln in Visual Studio.
2. Build project.

### Use

1. Prepare a folder with VSDX files.
2. Launch the project.
3. Enter the path to the folder with VSDX files in the console that has started.
4. Go to the folder with VSDX files, where the BPMN folder was created, which contains the program's output.
